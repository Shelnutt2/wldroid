# Vendored libdrm headers

Headers copied from libdrm 2.4.123 (source: `virgl/build/libdrm-install/include/`).
Used only for cross-compiling drm-shim — the runtime target (Debian inside proot)
has its own libdrm2 installed via Mesa dependencies.
